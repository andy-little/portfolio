money squared

An app I built for managing household bills and other shared costs.

Users first create a household. Members of this household can add their transactions to the group. Members can also:
  • view their entire transaction history
  • easily see who owes who what
  • settle debts
  • edit/delete transactions not yet settled
In this project I created my own paginator api which, unlike Django's, can be used asynchronously.
